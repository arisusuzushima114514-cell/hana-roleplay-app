# Hana

Hana 是一个 Android 平台上的多功能 AI 对话应用，核心方向包括角色扮演、多角色群聊、角色卡导入导出、图片生成、长期记忆和联网搜索。

这个仓库当前提供的是适合公开发布到 GitHub 的项目副本：已经移除了本地构建产物、IDE 缓存、本机配置文件和不适合直接公开的杂项目录。

## 碎碎念

大家好，我是万花。

这是我第一次认真做这一类 App。严格来说，我其实根本不懂代码，连很多项目文件该放哪里、缺什么配置、构建链路怎么跑都不明白。我会的事情很有限，基本只是把 Android Studio 连上雷电模拟器，然后一点点调试、看结果、再继续改。

之前因为期末作业，我按要求让 AI 做过一些音乐播放器、记事本、管理器之类的小项目。再加上我平时本来就会玩一些类似酒馆、Chatbox 这种带角色人格设定的 App，所以到了上周五晚上，我突然很强烈地觉得：我为什么不能试着自己做一个？

我当然知道市面上已经有很多成熟作品了，但我还是想试试看。

从 7 月 10 日周五晚上，到 7 月 14 日周二这几天里，我一边用 OpenCode，一边用 Codex，大概断断续续做了二三十个小时。中转站额度也确实烧了不少，差不多花了一百多块。过程基本就是一边做、一边测、一边修，很多东西都是靠一点点试出来的。

但说实话，这个过程让我很开心。

看着一个原本只是脑子里的想法，慢慢长成一个真的能装到手机里、能点开、能对话、能跑起来的 App，这种感觉真的很好。到现在为止，我自己手机里大概已经装了五六十个不同阶段的安装包。虽然现在很多地方还比较糙，也还只是一个雏形，但整体已经有一点我想要的样子了。

我把它放到 GitHub，一方面是想继续记录和迭代这个过程，另一方面也是想感谢 AI 时代。因为如果放在以前，像我这种几乎完全不懂代码的人，大概很难真的一步步做出一个自己想要的 App。但现在，我可以靠不断测试、不断调整、不断提需求，把脑子里的东西慢慢落下来。

这个作品现在还远远不完善，但它对我来说已经不只是一个练手项目了。

它是我自己一点点做出来的雏形。

未来我会继续更新它。

## 项目特性

- 角色对话（带内心想法、关系推进与状态记录）
- Persona 人格预设（主对话全局角色设定）
- 多角色群聊
- 角色卡导入与导出
  - JSON
  - PNG 角色卡（带 metadata）
- 场景捕捉生图（自动提取角色与场景，POV 视角生图）
- 图片生成
- 长期记忆
- 联网搜索开关
- 模型联网指示灯
- 角色头像 / 封面图自定义

## 开发建议

- 优先使用单独测试服务商验证联网 / 图像 / 多角色逻辑
- 发布前检查默认配置是否为空，避免把个人使用环境误带入仓库
- 角色卡 PNG / JSON 兼容性建议用多来源样本回归测试
