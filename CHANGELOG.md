# Changelog

## v1.7.5 (2026-07-18)

### 回复质量大幅优化（重要）
- 锚点新增「剧情回应要求」：强制逐一回应所有剧情点，不再只读最后两个
- 风格样本精简（8条→5条，每条400字→300字），释放 ~1000 tokens 给用户消息
- 上下文压缩阈值 30轮→10轮，让摘要机制默认生效，防止 token 溢出
- 单条消息截断 16000字→8000字，防止超长消息撑爆上下文
- 历史摘要优化为结构化事件列表，参考 OpenTavern 总结机制

### Bug 修复
- 角色卡导入头像空白：PNG角色卡导入时自动保存图片为本地头像
- 更新推送弹窗重复弹出：增加 hasShownThisSession 守卫，每次启动只弹一次

### 交互优化
- 用户消息支持一键复制（三点菜单），与角色消息一致
- 蓝色气泡内选中文字可见性修复：高亮色改为半透明白色

---

## v1.7.4 (2026-07-17)

### 长对话防退化（根治）
- 每次回复前强制注入角色卡锚点，角色身份始终在模型注意力窗口内
- 锚点四段式结构：内心想法指令 → 角色身份 → 风格样本 → 禁止事项
- 内心想法指令前置到锚点最前面，要求精炼（一两句话戳核心）
- 提取角色真实回复和内心想法作为格式示例
- 上下文压缩：超过30轮自动摘要，保留最近30轮完整内容

### 风格样本大幅增强
- 从对话开头提取高质量回复作为风格参照（前8条，每条400字）
- 额外注入角色开场白作为风格样本（500字）
- 避免人机化回复污染风格，保持角色一致性

### 角色对话界面精简
- 顶部栏精简为单行，冗余功能（模型设定、创作预设、联网、面板等）收入三点菜单
- 底部输入栏紧凑化：缩小圆角（24→20dp）、阴影（6→4dp）、内边距
- 键盘弹出时对话区域不再被挤压（imePadding 修复），可正常阅读回复

### 更新推送系统
- 新增"更新日志"弹窗，每次更新后首次打开 App 自动弹出
- "不再显示"选项：勾选后同版本不再弹出，下次更新自动弹出新版本日志
- 版本号自动同步到启动页和设置页
- 弹窗重复弹出 Bug 修复：增加 hasShownThisSession 守卫，每次启动只弹一次

### 多机型适配
- 参数面板支持垂直滚动，小屏设备不裁切按钮
- Slider 步数优化，低端设备流畅拖动
- 上下文轮数范围统一为 1~999
- 参数保存后即时同步 UI，不再显示旧值

### 其他
- 默认最大回复长度 4096 → 8192 tokens
- 深色模式配色优化
- buildConfig 启用，版本号可在代码中读取

## v1.7.3 (2026-07-17)

### 长对话防退化
- 角色锚点注入频率从每8轮提升至每5轮，锚点内容加入行为约束（禁止出戏表述、强制输出内心想法）
- 上下文压缩机制：超过20轮对话时自动压缩旧对话为摘要，只保留最近20轮完整内容，防止token窗口溢出导致角色卡被截断失忆

### Bug 修复
- 角色参数保存后即时同步UI状态，解决保存后重开面板仍显示旧值的问题
- 删除角色卡/清除对话时彻底清理好感度残留（DataStore + Room 双重清理）
- 停止生成按钮状态修复：try-finally 确保 isSending/streamingAssistant 始终重置

### 多机型适配
- 参数面板 Column 添加 verticalScroll，小屏设备不再裁切保存按钮
- Slider 移除过多 steps（997→0），低端设备滑动更流畅
- 上下文轮数 Slider 范围统一为 1~999（主对话面板从 1~50 扩展）

### UI 优化
- 默认最大回复长度从 4096 提升至 8192 tokens，角色输出更长更细致
- 深色模式配色优化：背景 #0A0B12、表面 #141520、主色 #A78BFA、AI气泡 #1E2036

### 技术优化
- 长对话滚动从 animateScrollToItem 改为 scrollToItem，进入对话瞬间到底部
- 消息气泡移除 AnimatedVisibility 入场动画，消除历史消息"从上往下读档"感

## v1.7.2 (2026-07-16)

### 新功能
- 场景捕捉生图：角色卡对话右上角新增「场景捕捉（测试）」按钮，自动提取角色外貌与当前对话场景，生成 POV 视角提示词，生图结果直接嵌入聊天对话流中显示，带加载动画和进度提示
- 模型联网指示灯：模型选择器新增绿色/灰色圆点 + "联网"/"离线"标签

### UI 优化
- 角色卡聊天界面布局重构：将「角色模型设定」从底部移至顶部 FlowRow，回复栏简化为紧凑单行（输入框 + 发送按钮），圆角 30dp→24dp、阴影 10dp→6dp、padding 缩小
- Persona 编辑器字数限制解除：移除 2000 字上限

### Bug 修复
- Persona 优先级修复：Persona 预设不再被模型版本声明覆盖，确认身份时不再出现"我是 DeepSeek"
- 角色卡自动滚动：进入角色卡对话时自动滚动到最新消息（底部）
- 场景捕捉重复点击防护：生成中按钮禁用，防止触发多个并行生图请求
- 切换角色清除生图残留：切换角色卡时自动清空上一个角色的场景捕捉结果
- 内心想法消失修复：系统提示词从软性「需要内心时」改为强制「每轮回复末尾必须」
- 角色参数设置保存修复：温度、Top-P、Max Tokens、上下文轮数等参数正确持久化，不再自动返回默认值
- Persona 沉浸过度修复：优化 Persona 系统提示词，角色性格只影响语气而非内容，即使傲娇/吃醋/高冷也能完整回答问题

### 技术优化
- `extractThinking` 正则改为行锚点匹配，防止误匹配正文中的 thinking/response 单词
- 回复栏动画改用 `graphicsLayer { scaleX/scaleY }` 替代 `Modifier.scale()`
